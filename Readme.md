Thank you for choosing **Erwin**! This Privacy Policy outlines how we collect, use, store  when you use the Erwin. Discord bot. By using Erwin, you agree to the practices described in this policy.

 # 🗂️ Data Collection

Erwin is a free-to-use Discord bot that offers powerful features like automod, anti-nuke, welcome messages, autoroles, Ticket System and many more. To function automod or other features properly, Erwin collects and stores non-personal data:

 **Server IDs** – to manage configurations on a per-server basis.
 **User IDs** – used for automod logs, punishments, blacklist tracking, and role systems and moderation commands.
 **Channel & Role IDs** – for handling welcome channels, log channels, Ticket Channels and automod targets.
 **Message Content** – temporarily analyzed to detect spam, invites, banned words, etc. We **do not store** message content permanently.
 **Server Settings** – like thresholds, punishments, and toggleable features per module.

All collected data is securely stored.

 # 🛠 Use of Data

We use collected data solely for the following:

 Enabling and configuring Erwin’s core features.
 Executing moderation actions (mute, ban, warn, etc.).
 Improving stability, performance, and user experience.
 Providing debugging support and bug resolution.

**We never sell or share your data with third parties.**


 # 📜 Message Monitoring

Erwin monitors messages in real time to detect and respond to:

 Spam, flooding, or repeated content.
 Blacklisted keywords or banned links.
 Unwanted invites or scam messages.

This processing happens **in memory only** and is not stored or logged unless moderation action is taken (e.g., a warns or mute), in which case only the metadata is retained (not full message content).

 # 🔐 Access to Data

 Only **Server owners** can manage their server’s data.
 **Bot developers** may access server data strictly for bug fixing or improving features.

Access to sensitive information is controlled and monitored internally.

# 📦 Data Storage & Retention

 Server data is retained **as long as Erwin is present in the server.**
 If Erwin is removed, data is retained for **7 days** to support easy setup.
 Data may be **permanently deleted** upon request by a server owner.

# 🧹 Data Deletion

To request data deletion, please contact us with your **Server ID** and proof of server ownership. We'll delete your data from our systems.
